# Dice Table

**Онлайн-стол для бросков кубиков** с многопользовательскими комнатами, 3D-анимацией и режимом игры по очереди.

## Начало работы

### 1. Клонирование репозитория

Сначала скопируйте проект с GitHub на ваш компьютер:

```bash
git clone https://github.com/GRaf-NEET/dice.git
```

Затем перейдите в созданную папку проекта:

```bash
cd dice
```

Теперь вы находитесь в папке проекта и можете выбрать способ установки:

---

### 2. Установка и запуск

#### Вариант A: Установка на сервер (Linux)
```bash
# Запустите из корневой папки проекта
# В ней должны быть install.sh и файлы проекта
chmod +x install.sh && sudo ./install.sh
```
Скрипт автоматически установит Python, Node.js, Nginx и запустит приложение.

#### Вариант B: Локальный запуск для разработки
```bash
# Создаем виртуальное окружение Python
python3 -m venv venv

# Активируем виртуальное окружение
# Для Linux/Mac:
source venv/bin/activate
# Для Windows:
# venv\Scripts\activate

# Устанавливаем зависимости
pip install fastapi uvicorn

# Запускаем сервер
uvicorn main:app --host 0.0.0.0 --port 8000
```

После запуска приложение будет доступно по адресу: `http://localhost:8000`

## Использование

### Для локальной игры в одной сети:
1. Запустите сервер одним из способов выше
2. Откройте приложение в браузере по адресу `http://localhost:8000`
3. Выберите режим "Онлайн"
4. Создайте комнату или введите код существующей комнаты

### Чтобы играть с друзьями:
1. **Убедитесь, что все игроки находятся в одной локальной сети (Wi-Fi)**  
   Например: домашняя сеть, офисная сеть или общая точка доступа

**2. Хост (тот, кто создаёт комнату) должен узнать свой IP-адрес в локальной сети:**
   - **На Windows:** откройте командную строку (cmd) → введите `ipconfig` → найдите строку **«IPv4-адрес»** (обычно выглядит как `192.168.x.x`).
   - **На Linux/Mac:** откройте терминал → введите `ifconfig` или `ip addr show` → найдите раздел вашего Wi‑Fi или Ethernet‑адаптера и строку **`inet`** (также обычно `192.168.x.x`).


3. Дайте друзьям адрес вида: `http://ВАШ_IP:8000`
   Пример: `http://192.168.1.5:8000`

4. Друзья открывают этот адрес в браузере и присоединяются к вашей комнате

### Важно:
- Для игры через интернет (из разных сетей) потребуется настройка переадресации портов на роутере или размещение на хостинге
- По умолчанию приложение работает только в локальной сети
